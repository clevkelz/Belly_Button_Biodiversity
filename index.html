Belly Button Biodiversity Dashboard

Use the interactive charts below to explore the dataset

These visualizations have been developed to break down the belly button bacterial data for each volunteer. Ultimately, this should help identify the top 10 bacterial species in volunteers' belly buttons. That way, if Improbable Beef identifies a species as a candidate to manufacture synthetic beef, this information can be used to identify whether that species is present in some form in people's navels.
